AI-Powered Sustainability Dashboard for Green Logistics

Key Features

AI Recommendations: Suggests ways to reduce emissions and improve efficiency.

Blockchain Reporting: Ensures transparent, tamper-proof records of sustainability efforts for compliance.

Orrery Web App

This is an interactive web-based orrery that visualizes the orbits of celestial bodies, including planets, Near-Earth Asteroids (NEAs), Near-Earth Comets (NECs), and Potentially Hazardous Asteroids (PHAs). Using real-time NASA data, this educational tool will help users explore the solar system and understand celestial mechanics, built with Python and Flask for the backend and JavaScript (Three.js) for 3D rendering.


Key Features

Real-time Data Integration: Fetch and display accurate positions and orbits of celestial bodies using NASA APIs.

Interactive Visualization: Enable users to zoom, rotate, and navigate the solar system.

Orbital Path Visualization: Show colored trajectories for each celestial body.

Timeline Control: Allow users to adjust simulation speed and view positions over time.
